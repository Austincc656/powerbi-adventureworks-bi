# Power BI Business Intelligence Project – AdventureWorks

This repository documents an end-to-end Power BI Business Intelligence project
using the AdventureWorks dataset. The project follows a real-world BI workflow,
from raw data preparation to interactive dashboard design.

## Project Objectives
- Transform raw CSV data into clean, analysis-ready tables
- Build a relational data model for business analysis
- Create KPIs and metrics using DAX
- Design interactive dashboards for decision-makers

## Tools & Technologies
- Microsoft Power BI Desktop
- Power Query (ETL & data transformation)
- DAX (Data Analysis Expressions)
- Relational data modeling (star schema)

## Project Structure
- `data/` – Raw source data files
- `powerbi/` – Power BI Desktop (.pbix) file
- `screenshots/` – Visual evidence of transformations and reports
